# ChenListen 晨听 · 网站主旨 · 2026-06-11 口述，2026-06-14 更新

## 一句话
围绕音乐、声音、架子鼓的创作合集——表达**热爱生活、发现生活中的美、感受生活、享受大自然的馈赠**。

## 内核
快节奏让我们这代人可以一辈子活成三辈子，但我们的质量不变——
**我们爱慢生活，爱人与人的沟通。**

网站要做的不是"展示作品"，是把这种生活态度寄给打开它的人。

## 品牌（2026-06-14 定）
- **主品牌 = ChenListen 晨听 = 作者本人**：晨听 = 晨(morning) + 听(listen) = ChenListen。主页、左上 logo、每张明信片的 FROM 落款都用「晨听 · ChenListen」。
  （注意：名字读作「晨听」；logo 图形概念历史上叫「听叶」= 鼓棒开花成叶、叶尖发出声波 = 敲击长出自然，是图形不是名字，别再把名字写成"听叶"。）
- **DrumFeather 鼓羽 收窄**为「架子鼓绘本」那一个项目的名字，不再当全站伞品牌。
  理由：笔名是作者身份，作品横跨 jazz/声音/鼓房/氛围音乐，用"听"当伞比"鼓羽"更兜得住。

## 封面原则（2026-06-14 定）
- 隐喻不换：**明信片 = 慢生活/人与人沟通的前数字时代实物**（手写、要寄、要等、写给一个具体的人）
  = "年代味/人味"那一轴；活窗 / 翻面 / 以后 hover 出声 = "现代技术吸引年轻人"那一轴。隐喻=主旨本身。
- 灵魂是 **FROM / TO**：每张寄给一个具体的人的时刻（"说不出话的那个晚上"），不是穿戏服的作品网格。
- **墙上不留空卡**：页面是给别人看的，空信封（虚线"贴邮票处"）让人失望。失望的根子不是"没做完"，是"空"。
  在做的不藏——用**真实画面 + 一句真话**，绘本放真剧照标"连载中"；真一张画面都没有的才暂时下墙。
